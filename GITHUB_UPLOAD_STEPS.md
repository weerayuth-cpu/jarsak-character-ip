# GitHub Upload Steps

ใช้ไฟล์ทั้งหมดในโฟลเดอร์นี้อัปเดต repository GitHub Pages เดิม

## วิธีอัปโหลดผ่านหน้าเว็บ GitHub

1. เปิด repository เดิมใน GitHub
2. กด `Add file` > `Upload files`
3. ลากไฟล์และโฟลเดอร์ทั้งหมดจาก `opendurian-english-tiktok-ip` ไปวาง
4. ถ้า GitHub ถามเรื่องไฟล์ซ้ำ ให้เลือกแทนที่ไฟล์เดิม
5. ใส่ commit message:

```text
Update OpenDurian English TikTok character IP site
```

6. กด `Commit changes`

## GitHub Pages

ถ้าเว็บยังไม่ขึ้น ให้ไปที่ `Settings` > `Pages`

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/root`

## ไฟล์สำคัญ

- `styles.css`
- `script.js`
- `README.md`
- `.nojekyll`
- `assets/images/shots/`
- `assets/images/green-screen/`
